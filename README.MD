## Запуск:
Для Запуска приложения Вам необходим установленный docker compose.
Что бы запустить сервис создайте и заполните файл `.env` (по образцу из `.env.example`) и выполните:
```bash
docker compose up -d
```

## Миграции

### Создание миграции
```bash
php commands/migrate.php make create_users_table
```

### Запуск миграций
```bash
php commands/migrate.php run
```

### Откат последней миграции
```bash
php commands/migrate.php rollback
```

### Откат 3-х последних миграций
```bash
php commands/migrate.php rollback 3
```

### Полный рефреш (откат всех и накат заново)
```bash
php commands/migrate.php refresh
```

### Генерация Сидов
```bash
php commands/seed.php
```
